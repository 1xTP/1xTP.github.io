1. Overview

A couple of days ago, my HackTheBox VIP subscription ran out, and since I lost my easy access to their Active Directory environments, I decided it was time to build my own AD homelab in VirtualBox. This actually works out better, because now I can keep the environment forever and break it whenever and however I want.

My goal is simple: Setup an AD environment that I can make vulnerable and hack in any way I want to.

2. Network Overview

I kept the setup pretty small to save resources. I have 3 virtual machines in the network. They are all connected through a NAT network in VirtualBox. This keeps them isolated from my home network while allowing them to still talk to each other. Currently only the Windows 11 client is joined to the domain. The Kali box is just the standard weekly image from Offensive Security

3. DC setup

Getting the Domain Controller running on Windows Server 2022 was actually easier than I expected. I set the IP to be static as that was the first warning I saw. Once the networking was setup, I installed the AD Domain Services and DNS roles. After that, I just promoted the server to be the Domain Controller for my new domain, trashpanda.local.

4. Windows Client Setup

Installing the Windows 11 client is pretty standard, it's just like setting up a normal PC. After I connected this client to the domain, it was finished and I was able to log in.

5. Automating User Creation

I started creating users manually, but I realized about three users in that it was going to be incredibly tedious to do this every time I reset the lab.

Instead of wasting time , I wrote a PowerShell script to do it for me. It reads a list of names from a CSV file and automatically creates the users, assigns them to the right departments (like HR or Sales), and sets their passwords.

6. Conclusion and Next Parts

Now that my "Gold Image" is set up and working, the boring part is over.

In the next part, I’m going to start breaking things. I plan to set up some Service Accounts with weak permissions so I can practice Kerberoasting with my Kali box.
